Zing Expression

A zing expression is exactly like a javascript expression, however to access your tag's environmental variables, use $$
For example home.js could include a variable like $env.helloWorld = "hello world";
To link this to an html element, you'd use a zing expression i.e link="$$helloWorld"

click

Put a click attribute to add a click event listener.
You can use $$ to use $env variables.

link

link will set the innerHTML to the zing expression.
